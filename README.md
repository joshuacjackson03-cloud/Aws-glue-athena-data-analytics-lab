# AWS Glue and Athena Data Analytics Lab

## Overview
This project demonstrates how to prepare and analyze data using AWS Glue and Amazon Athena.

## AWS Services Used
- Amazon S3
- AWS Glue
- AWS Glue Crawlers
- AWS Glue Data Catalog
- Amazon Athena

## Lab Objectives
- Upload data to Amazon S3
- Discover schema using AWS Glue Crawlers
- Create metadata tables in the Glue Data Catalog
- Query data using Athena

## Architecture

Data Source
↓
Amazon S3
↓
AWS Glue Crawler
↓
Glue Data Catalog
↓
Amazon Athena

## Skills Demonstrated
- Serverless Analytics
- Data Lakes
- SQL Queries
- Metadata Management
- AWS IAM Permissions

## What I Learned
- How Athena queries data directly from S3
- How Glue Crawlers automatically discover schema
- How the Glue Data Catalog stores metadata
- Basic data analytics workflows in AWS
